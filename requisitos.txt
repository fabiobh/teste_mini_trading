Teste Técnico – Plataforma de Trading (Mini Binance)

Objetivo
Construir uma aplicação simples de compra e venda de ativos com backend em Laravel e app em React Native.

Escopo do Projeto
1. Autenticação
1.1 Registro de usuário
1.2 Login com email/senha
1.3 Endpoint protegido /me
1.4 Uso de Laravel Sanctum ou JWT

2. Carteira (Wallet)
2.1 Saldo inicial: R$ 10.000 e 0 BTC
2.2 Endpoint GET /wallet
2.3 Retornar saldos do usuário

3. Mercado (Preço fake)
3.1 Preço dinâmico do BTC
3.2 Range sugerido: 200k–300k
3.3 Endpoint GET /market/btc

4. Compra e Venda
Compra BTC:
4.1 POST /trade/buy
4.2 Validar saldo
4.3 Converter valor
4.4 Atualizar carteira
Venda BTC:
4.5 POST /trade/sell
4.6 Validar saldo BTC
4.7 Converter valor
4.8 Atualizar carteira

5. Histórico de Transações
5.1 GET /transactions
5.2 Listar tipo, valor, preço e data

React Native (App)
1 Tela de login/registro
2 Dashboard com saldos e preço
3 Tela de trade (comprar/vender)
4 Tela de histórico

Critérios de Avaliação
1 Backend (40%)
2 Regras de negócio (25%)
3 Mobile (20%)
4 Código e organização (10%)
5 Extras (5%)

Diferenciais
1 Uso de cache (Redis)
2 Testes automatizados
3 Controle de concorrência
4 Docker
